# 防汛值班值守制度（通用版草案）

## 适用范围

本制度适用于本级防汛指挥机构办公室、水务部门防汛值班室、工程运行管理单位值班岗位，以及在防汛响应期间参与联合值守的成员单位。制度按“常态值守 + 响应升级加密值守 + 重大险情战时值守”三种状态执行，确保雨情、水情、工情、险情、灾情信息不断线、不断档。依据国家防总值班规定，值班实行领导带班和工作人员值班相结合的全天 24 小时制度；进入主汛期、超警戒水位、发生较大险情灾情或启动响应时，应提高带班层级并加密值守。

## 组织与岗位

| 岗位 | 主要职责 |
|---|---|
| 带班领导 | 审定重要信息，决定会商、督办、叫应和升级报送事项 |
| 值班长 | 统筹当班工作，复核信息口径，组织交接班 |
| 值班员 | 接收、登记、核实、转办各类来电来文和平台告警 |
| 技术支撑岗 | 汇总雨量、水位、流量、库容、泵站负荷、工程运行状态 |
| 通信保障岗 | 保障值班电话、专网、视频会商、短信平台、应急电源可用 |

建议带班领导、值班长、值班员、技术支撑岗形成最小闭环。启动 III 级及以上响应时，可按本地预案增加宣传、交通、住建、城管、应急、气象、水文等单位联合值守。

## 工作规则

值班人员必须在岗在位，电话、对讲、视频、专网、移动终端保持畅通，不得擅离岗位。

重要信息实行“即收、即登、即核、即报、即跟踪”，不得以“待核实”为由压报、缓报。

交接班必须面对面进行，交清未办事项、关注水库河道、在途队伍、待回复口径和上级督办事项。

值班日志、来电记录、调度指令、会商纪要、图片视频、签报材料应同步留痕归档。

对群众来电、基层求援、工程告警和媒体关注事项，应区分“咨询件、一般事项、重要事项、突发事件”四类分别处置。

## 时限要求

| 事项 | 处置要求 |
|---|---|
| 预警、险情、灾情、工程异常来电 | 当班立即登记并报告值班长 |
| 可能影响人员安全的重要信息 | 立即报告带班领导并启动核实 |
| 突发重大险情灾情 | 允许电话先报，随后书面补报 |
| 县级以上防指确认突发重大险情灾情 | 按国家规定 1 小时内向上级和国家防总办公室报告 |
| 大江大河重要堤防、重要涵闸及大型和防洪重点中型水库重大险情 | 立即报告 |
| 日常值班摘要、日报、夜报 | 按 {{本地报送时点}} 执行 |

首报、续报和终报的时限，以本级信息报送制度为准；但涉及国家规定的重大险情灾情，不得用本地时限替代国家刚性要求。

## 上报与通知对象

一般事项上报值班长；重要事项上报带班领导；涉及响应启动、群众转移、工程调度、停课停工停运建议的事项，同步通知 {{本级防指负责人}}、{{水务/应急/气象/水文联络人}}、{{工程管理单位}} 和 {{属地政府}}。涉及公众风险的信息，未经授权不得自行对外发布。

## 风险提示

值班制度最容易失效的环节是“只接不办、只报不跟、口头交接、无留痕”。任何班次不得出现漏接电话、重复报送口径不一致、工程险情无人盯守、转移指令未闭环反馈等问题。主汛期和关键过程，原则上不安排与值班无关的会议、外出和休假顶岗不清。
