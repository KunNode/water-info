# 城区内涝处置流程（演示版）

## 一、适用范围

本流程用于防汛演示知识库，适用于短历时强降雨、排水管网满负荷、河道顶托、泵站排涝能力不足导致的城区积水和内涝处置。重点对象包括下穿通道、低洼小区、地下商场、地铁口、医院、学校、养老院、变电站、供水厂、交通枢纽和主干道路。

## 二、处置目标

1. 优先保障人员生命安全。
2. 快速封控危险积水点和地下空间。
3. 组织排涝、清障、交通疏导和群众转移。
4. 保障医院、学校、供电、供水、通信等重点设施。
5. 及时发布公众提示，减少车辆涉水和人员围观。

## 三、内涝风险分级

| 等级 | 判定参考 | 处置重点 |
|---|---|---|
| 一般积水 | 积水深度小于 15 厘米，短时可退 | 巡查、清沟、疏通雨水口 |
| 较重积水 | 积水 15 至 30 厘米，影响行人和小型车辆 | 设置警示、安排抽排、疏导交通 |
| 严重内涝 | 积水 30 至 50 厘米，车辆通行困难 | 封控道路、组织排涝、转移地下空间人员 |
| 危急内涝 | 积水超过 50 厘米或快速上涨，人员被困 | 救援优先、断电封控、扩大转移 |

积水深度只是参考，若存在强水流、井盖缺失、触电风险、地下空间进水、危化品风险，应按更高等级处置。

## 四、处置流程

### 1. 接报与定位

接到积水信息后，应记录时间、地点、积水深度、范围、是否影响交通、是否有人员被困、是否进入地下空间、现场联系人和照片视频。系统应优先匹配历史易涝点、泵站服务范围、雨水管网和附近重点设施。

### 2. 初判与派单

| 场景 | 主办单位 | 协同单位 |
|---|---|---|
| 道路积水 | 城管或排水部门 | 公安交警、交通 |
| 下穿通道积水 | 交通或城管 | 公安交警、消防、排水 |
| 小区内涝 | 属地街道 | 住建、物业、消防 |
| 地下空间进水 | 属地街道或行业主管 | 消防、电力、公安 |
| 重点设施受威胁 | 行业主管 | 应急、水务、电力、通信 |

### 3. 现场封控

积水超过 30 厘米或无法确认井盖、路基、供电安全时，应立即设置警戒。下穿通道、地下车库、地铁口、地下商场进水时，应第一时间组织人员撤离，并联系电力部门评估断电。

### 4. 排涝处置

排涝措施包括清理雨水口、开启排水闸、启动泵站、调派移动泵车、铺设临时排水管、打开临时排涝通道、清除漂浮物和垃圾。外河高水位顶托时，应防止倒灌，优先采用泵排。

### 5. 交通疏导

公安交警和交通部门应根据积水深度和道路等级实施限行、绕行或封路。公交、地铁、客运场站应同步调整运行计划。危险路段应设置反光锥、警戒带、提示牌和夜间照明。

### 6. 群众转移

当积水进入居民楼一层、地下室、养老院、学校、医院或危旧房时，应组织人员转移。转移应优先照顾老人、儿童、病人、残障人员和孕妇。安置点应准备饮水、食品、照明、医疗和登记台账。

## 五、重点场景

### 1. 下穿通道

下穿通道水位达到警戒线或排水泵故障时，应立即封闭入口，禁止车辆和行人进入。已进入车辆应在安全条件下组织人员撤离，严禁盲目推车、涉水通行。需要救援时由消防或专业救援队伍实施。

### 2. 地下车库

发现倒灌或雨水倒排时，应通知物业开启排水泵、关闭防火卷帘外的危险通道、组织人员撤离。电力部门确认前不得擅自送电。车辆财产抢救不得优先于人员安全。

### 3. 医院学校

医院应优先保障急诊、ICU、配电房、供氧设施和药品库。学校应暂停户外活动，必要时延迟放学或组织留校避险。涉及停课、停运、停诊等事项应由主管部门统一发布。

### 4. 变电站和通信机房

应设置挡水设施和临时排水，安排专人巡查电缆沟、配电室和机房入口。出现进水或触电风险时，应立即封控并通知专业人员处置。

## 六、信息发布

公众提示应包括积水点位置、封路绕行路线、不要涉水通行、远离井盖和带电设施、地下空间人员撤离、车辆不要进入下穿通道等内容。信息发布应及时更新，积水消退和交通恢复后也应发布解除提示。

## 七、处置反馈

现场单位应滚动反馈：

1. 积水深度和面积变化。
2. 排涝设备数量和运行状态。
3. 封控道路和绕行路线。
4. 人员被困、转移和安置情况。
5. 需要增援的队伍、泵车、发电机、照明设备。
6. 预计恢复时间。

## 八、恢复与复盘

积水消退后，应检查道路塌陷、井盖缺失、电气进水、地下空间污染、管网堵塞和泵站设备损坏。对反复积水点应形成复盘清单，包括积水原因、排涝瓶颈、责任单位、整改措施和完成时限。

## 九、演示用关键词

城区内涝、积水点、下穿通道、地下空间、移动泵车、雨水口、排涝泵站、交通封控、群众转移、触电风险、道路绕行、易涝点。
