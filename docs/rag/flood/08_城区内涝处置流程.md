# 城区内涝处置流程（通用版草案）

## 适用范围

本文件适用于城市低洼区域、易涝点、下穿隧道、地铁站口、地下商场、地下车库、老旧小区、城乡接合部和生命线工程周边的内涝应急处置。国务院办公厅关于加强城市内涝治理的实施意见和住房城乡建设部、应急管理部通知均要求落实省负总责、城市具体落实的体制，公开责任人，明确启动条件，强化会商研判、跨区域支援和地下空间等关键部位防范。

## 处置流程

### 雨前准备

清理雨水口、格栅、泵站前池和关键排水通道。

检查闸泵、应急抽排车、移动电源、挡水板、沙袋和警示设施。

对地下车库、地铁站口、下穿隧道、地下商业体和低洼院落布置防倒灌措施。

对易涝点、危房、边坡、工地、户外广告和路树开展风险排查。

广州防暴雨内涝响应中明确要求在雨前雨后清扫雨水口表面垃圾、布防抢险队伍、检查地下空间防倒灌措施并排查供配电设施风险。

### 雨中处置

对重点易涝点实行路面巡查和视频巡看双轨运行。

对积水点立即采取抽排、围蔽、警示、交通引导或封闭。

对低洼易涝区、危房、边坡、简易工棚、地下空间等危险区域人员实施撤离。

对供水、供电、供气、通信等生命线设施加强巡查，必要时采取停电防触电等措施。

对地铁站口、地下车库、下穿隧道等高风险部位严防倒灌。

广州公开预案在内涝三级、二级响应中要求组织排查房屋、道路、桥梁、地下空间、涵隧、地铁等设施，做好低洼区人员安全防范、交通管控和地下空间防倒灌。

### 雨后恢复

复核积水消退情况，逐点销号。

排查道路、桥涵、泵站、供电设施和地下空间是否可恢复运行。

清运淤积、消杀清洁、统计受损。

形成易涝点处置复盘，更新风险点台账和下轮布防方案。

住建部、应急管理部通知明确要求将灾后复盘与预案修订结合起来。

## 重点场景处置表

| 场景 | 首要动作 | 升级条件 |
|---|---|---|
| 下穿隧道 | 警示、封控、抽排、交通分流 | 水深接近车辆涉水危险值 |
| 地铁站口/地下商场 | 挡水、防倒灌、疏散人流 | 积水持续上涨或外部道路倒灌 |
| 地下车库 | 禁止车辆驶入、断电、抽排 | 已有车辆受困或人员被困 |
| 低洼小区 | 转移老人儿童病患、发放告知 | 夜间持续强降雨、排水失效 |
| 城市主干道积水 | 布设警示、交警联动、抢排 | 影响公交、医院、学校通行 |

## 风险提示

城区内涝最危险的不是“水多”，而是“倒灌 + 触电 + 夜间 + 交通误判 + 地下空间滞留”叠加。任何时候不得允许社会车辆、行人盲目通过深水区；不得在未确认电气安全前组织涉水排障；不得忽视老人、儿童、残障人员和独居住户的转移提醒。
