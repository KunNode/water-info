# 通知通报模板与首报/续报模板（通用版草案）

## 适用范围

本文件用于快速生成响应启动通知、险情灾情首报、续报、终报和复盘通报。模板字段保留了国家防汛信息报送制度中的核心要素，兼容值班报送、会商纪要、短信告警和正式公文摘报场景。首报、续报和信息发布实行分级负责，紧急情况下允许先电话报告、后书面补报。

## 使用规则

模板中的 {{字段}} 为待填项，不得留空后外发。

首报重在“事实 + 趋势 + 已采取措施 + 需支持事项”。

续报重在“变化量 + 新风险 + 新动作 + 新需求”。

对外发布口径必须与本级防指授权口径一致。

形成正式文件时，应补齐签发人、时间、密级和附件清单。

## 风险提示

模板是“最小字段集”，不是“最少工作量”。凡是已经掌握但未写入的信息，后续都可能成为追责盲点。首报不要堆砌背景，续报不要重复旧话，终报不要只写成绩不写问题。涉及公众风险的信息发布，必须遵守本级防指授权。
