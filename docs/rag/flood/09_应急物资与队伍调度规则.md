# 应急物资与队伍调度规则（演示版）

## 一、适用范围

本规则用于防汛演示知识库，适用于防汛抢险队伍、工程机械、排涝设备、砂石料、编织袋、土工布、照明通信、救生器材、生活保障物资的调度。规则服务于系统在预案生成、资源推荐、调度派单和抢险复盘中的知识检索。

## 二、调度原则

1. **就近优先**：优先调用距离风险点最近、到场最快的队伍和物资。
2. **专业匹配**：堤防险情调用水利抢险和工程技术队伍；人员被困调用消防救援；内涝调用排水和移动泵车。
3. **生命优先**：人员搜救、群众转移、医疗救护优先于财产抢救。
4. **分级审批**：一般物资由值班长调派，跨区大批量物资由防指或授权领导批准。
5. **闭环管理**：调拨、出库、运输、到场、使用、回收、补库全流程留痕。

## 三、队伍类型

| 队伍类型 | 适用场景 | 主要装备 |
|---|---|---|
| 水利工程抢险队 | 堤防管涌、渗漏、滑坡、闸站故障 | 砂袋、土工布、抛石、测量设备 |
| 消防救援队 | 人员被困、涉水救援、地下空间 | 冲锋舟、救生衣、绳索、照明 |
| 排水抢险队 | 城区内涝、泵站故障、地下车库进水 | 移动泵车、水带、发电机 |
| 交通保障队 | 道路中断、桥涵封控、清障 | 清障车、警示标志、装载机 |
| 电力抢修队 | 变电站进水、线路故障、临时供电 | 发电车、抢修车、绝缘工具 |
| 通信保障队 | 基站中断、现场通信不畅 | 应急通信车、卫星电话、对讲机 |
| 医疗保障队 | 安置点医疗、伤员救治 | 急救车、药品、担架 |

## 四、物资分类

| 类别 | 物资示例 | 主要用途 |
|---|---|---|
| 抢险封堵 | 编织袋、砂石料、块石、木桩、钢管 | 堤防加固、子堤、围井 |
| 防渗反滤 | 土工布、无纺布、碎石、砂砾 | 管涌、渗水、反滤导排 |
| 排涝抽排 | 移动泵、泵车、水带、发电机 | 城区积水、地下空间排水 |
| 救生救援 | 救生衣、救生圈、冲锋舟、橡皮艇 | 涉水救援、人员转移 |
| 照明通信 | 移动照明灯、对讲机、卫星电话 | 夜间作业、现场指挥 |
| 生活保障 | 帐篷、折叠床、饮用水、食品、棉被 | 安置点保障 |
| 安全防护 | 雨衣、雨靴、安全帽、警戒带 | 现场人员防护和封控 |

## 五、调度触发条件

出现以下情况，应发起资源调度：

1. 响应启动后，需要队伍物资预置。
2. 堤防、水库、闸站、泵站出现险情。
3. 城区内涝影响交通、地下空间或重点设施。
4. 需要组织群众转移安置。
5. 现场请求专业设备、专家或增援力量。
6. 现有属地资源不足，需要跨区域支援。

## 六、调度流程

| 步骤 | 内容 | 责任人 |
|---|---|---|
| 需求确认 | 明确地点、险情类型、所需队伍物资、到场时限 | 值班长、现场负责人 |
| 资源匹配 | 根据距离、库存、专业能力、交通条件推荐资源 | 调度岗 |
| 审批派单 | 按权限审批，生成调度单 | 带班领导或授权人员 |
| 出库出动 | 物资出库、队伍集结、车辆装载 | 仓库和队伍负责人 |
| 在途跟踪 | 跟踪路线、预计到达时间、交通障碍 | 联络协调岗 |
| 到场确认 | 现场签收，反馈数量和状态 | 现场负责人 |
| 使用反馈 | 报告使用量、剩余量、效果和新增需求 | 现场信息员 |
| 回收补库 | 回收可用物资，统计消耗并补库 | 物资管理岗 |

## 七、资源匹配规则

1. 堤防管涌优先匹配砂石料、土工布、编织袋、水利抢险队和工程专家。
2. 漫顶风险优先匹配编织袋、移动挡水板、装载机、照明设备和转移队伍。
3. 城区内涝优先匹配移动泵车、水带、发电机、排水抢险队和交通封控力量。
4. 地下空间进水优先匹配消防救援、电力抢修、排水设备和警戒设施。
5. 水库泄洪影响优先匹配下游叫应人员、巡查队伍、交通管控和救生装备。
6. 夜间抢险必须同步匹配移动照明、通信设备和安全防护物资。

## 八、调度单字段

调度单应包含：调度编号、事件编号、申请单位、申请人、联系电话、目的地、险情类型、物资名称、数量、队伍名称、人数、车辆、出发时间、预计到达时间、审批人、接收人、签收时间、使用反馈、回收补库情况。

## 九、库存预警

系统应对关键物资设置库存预警。建议低于以下比例时提醒补库：

| 物资 | 预警规则 |
|---|---|
| 编织袋、土工布 | 低于常备量 40% |
| 砂石料、块石 | 低于常备量 50% |
| 移动泵、发电机 | 可用率低于 80% |
| 救生衣、救生圈 | 可用数量低于转移任务需求 |
| 照明和通信设备 | 可用率低于 90% |

## 十、运输与安全

物资运输应优先选择安全、通畅、避开积水和塌方风险的路线。大型机械进入堤顶、桥梁、窄路前，应评估承载能力。夜间运输应配备照明和引导车辆。危险区域装卸应设警戒线，禁止群众围观。

## 十一、复盘要求

每次调度结束后，应复盘响应时间、到场时间、物资匹配度、消耗量、损耗量、短缺情况和协同问题。对反复短缺物资、到场慢队伍、路线不畅点位，应纳入整改清单。

## 十二、演示用关键词

应急物资、抢险队伍、资源调度、编织袋、土工布、移动泵车、发电机、冲锋舟、救生衣、砂石料、库存预警、调度单、到场确认。
