# 水库调度与泄洪操作要点（通用版草案）

## 适用范围

本文件适用于承担防洪任务的水库汛期调度、实时泄洪、应急调度和下游预警工作。国家规定明确，水库汛期调度应坚持安全第一、统筹兼顾、兴利服从防洪、局部服从整体，实行统一调度、分级负责；调度执行单位应在汛前编制年度汛期调度方案，经有审批权限的调度管理单位批复后执行。

## 责任分工

| 角色 | 主要职责 |
|---|---|
| 调度管理单位 | 审批或下达调度方案、调度指令，监督执行 |
| 水库运行管理单位 | 执行调度指令，实施闸门和机组操作，留存记录 |
| 行政责任人 | 统筹协调属地防汛和群众避险事项 |
| 技术责任人 | 审核调度参数、风险判断、操作条件 |
| 巡查责任人 | 负责库区、坝体、泄洪设施和下游告警巡查 |

小型水库应落实行政、技术、巡查“三个责任人”；大中型水库应落实调度管理与调度执行双链条责任。

## 启动或适用条件

建议在以下情形下启动调度准备或实时调度：

预报未来来水显著增加，库水位接近 {{汛限水位}}。

上级调度管理单位下达预泄、错峰、拦洪削峰、联合调度指令。

坝体、闸门、启闭机、供电、通信、监测系统出现异常，需要调整运行方式。

下游河道安全泄量、下游防洪能力或下游在建工程条件发生变化。

水库为病险水库或限制运用水库，需要执行专门度汛方案。

国家水资源调度管理办法还明确提出，病险水库主汛期原则上不蓄水。

## 操作流程

数据确认：核对雨量、入库流量、库水位、闸门状态、机组状态和下游水位。

技术研判：比对年度汛期调度方案、实时调度指令、下游安全泄量和工程工况。

发令执行：一般以书面调度指令执行；紧急情况下可先电话下达并及时补发书面指令。

泄洪预警：出库流量或下游水位将明显变化时，按责任分工第一时间向下游政府、部门、单位和社会公众发布预警。

过程记录：逐时记录库水位、出入库流量、开闸孔数、机组工况、告警和指令执行情况。

反馈复盘：操作后向调度管理单位反馈执行情况；洪水过程结束后形成调度总结。

调度执行单位必须严格执行调度指令；如对指令有异议，在未接到新指令前仍应执行当前指令。遇紧急水情、工情危及工程安全时，可先行采取应急调度措施，并同步上报。

## 风险提示

严禁擅自变更汛限水位、擅自超权限调度、只调工程不做下游预警。泄洪不是单纯“开闸放水”，而是“调度指令 + 下游预警 + 工程巡查 + 过程记录 + 风险管控”的完整闭环。对病险水库、带病运行设施和通信供电不稳定工程，应预设人工操作和现场盯守方案。
