# 水库调度与泄洪操作要点（演示版）

## 一、适用范围

本文件用于防汛演示知识库，适用于中小型水库和具有防洪任务的调蓄工程。内容覆盖汛限水位控制、预泄、错峰、泄洪、巡查、预警、下游联动和信息报送。实际调度必须服从批准的水库调度运用计划和有调度权限单位的指令。

## 二、基本原则

1. **安全第一**：确保大坝、溢洪道、放水设施和下游群众安全。
2. **依法调度**：不得擅自超权限调度，不得无令泄洪或拒不执行调度令。
3. **预泄错峰**：根据预报提前腾库，尽量削减洪峰、错开下游高水位。
4. **信息先行**：泄洪前必须完成下游预警、叫应和人员避险。
5. **留痕可追溯**：调度令、闸门操作、监测数据、通知记录应完整归档。

## 三、关键水位概念

| 名称 | 含义 | 管理要求 |
|---|---|---|
| 汛限水位 | 汛期允许控制的最高蓄水位 | 汛期一般不得长期超过 |
| 正常蓄水位 | 非汛期正常运行水位 | 汛期不作为安全控制上限 |
| 警戒水位 | 需要加强监测和调度研判的水位 | 达到后加密巡查 |
| 设计洪水位 | 设计洪水条件下的高水位 | 接近时需高级别调度 |
| 校核洪水位 | 极端洪水校核条件下水位 | 接近时进入非常防洪状态 |

## 四、调度触发条件

出现以下情况之一，应组织调度会商：

1. 水库水位接近或超过汛限水位。
2. 未来 24 小时预报库区有暴雨或大暴雨。
3. 上游来水明显增加，入库流量持续上涨。
4. 下游河道已高水位运行，需要协调错峰。
5. 大坝、溢洪道、输泄水建筑物出现异常。
6. 上级防指或调度机构要求研判调度。

## 五、预泄操作要点

预泄用于在强降雨到来前腾出防洪库容。预泄前应完成：

1. 核实当前水位、库容、入库流量、出库流量和闸门状态。
2. 调取未来 6 小时、12 小时、24 小时降雨预报和上游来水预估。
3. 评估下游河道承泄能力和沿河风险点。
4. 向有调度权限单位申请或确认调度指令。
5. 通知下游乡镇、村社、施工单位、涉水景区和河道作业人员。
6. 记录通知时间、对象、反馈和未接通情况。

预泄应小流量、分阶段、可监测地实施，避免突然加大下泄流量造成下游风险。

## 六、泄洪操作流程

| 步骤 | 操作内容 | 注意事项 |
|---|---|---|
| 指令确认 | 核对调度令编号、签发单位、目标流量、开始时间 | 两人复核，不清楚不得执行 |
| 设备检查 | 检查闸门、启闭机、电源、备用电源、通信 | 异常先报告并提出替代方案 |
| 下游预警 | 完成电话、短信、广播、巡查叫应 | 重点通知河道内人员撤离 |
| 开闸操作 | 按规定顺序逐步开启闸门 | 严禁超调度令流量 |
| 过程监测 | 监测水位、流量、闸位、振动、渗流 | 异常立即停调或调整 |
| 续报反馈 | 向调度机构和值班室报告执行情况 | 包括开始时间、流量、水位变化 |
| 结束复核 | 按令关闸或调整至新工况 | 记录最终水位和设备状态 |

## 七、泄洪前预警范围

下游预警对象至少包括：

1. 沿河乡镇、村社和低洼住户。
2. 河道施工点、采砂点、临时便道和施工营地。
3. 涉水景区、露营地、农家乐和垂钓点。
4. 下游桥梁、漫水路、渡口和临河道路管理单位。
5. 下游水闸、泵站、排涝站和其他水工程。
6. 公安、交通、应急、消防、电力、通信等联动单位。

## 八、巡查检查

泄洪期间应加密巡查大坝、坝肩、坝基、溢洪道、输水涵洞、启闭设施、管理房、电源线路和下游消能设施。巡查重点包括渗漏、浑水、裂缝、塌陷、滑坡、异响、振动、闸门卡阻、启闭机过热、备用电源故障等。

发现异常应立即报告，并根据险情类型采取降低水位、减少泄量、加固坝坡、封控区域、调派专家等措施。

## 九、下游联动

水库泄洪可能叠加下游河道洪水和城市排涝压力。调度前应与下游防指、水文站、河道管理单位和闸站泵站沟通，确认下游控制断面水位、承泄能力、排涝泵站状态和人员转移情况。下游水位已接近警戒时，应优先会商错峰方案。

## 十、信息报送

泄洪首报应包含：水库名称、当前水位、汛限水位、入库流量、计划出库流量、开始时间、预计持续时间、下游影响范围、预警通知完成情况。续报应滚动报告水位、流量、闸位、设备状态、下游反馈和风险变化。

## 十一、禁止事项

1. 未经授权擅自泄洪。
2. 未通知下游即突然加大下泄流量。
3. 只发通知不确认下游接收。
4. 闸门操作无两人复核。
5. 设备异常时隐瞒不报。
6. 泄洪结束后不复盘、不归档。

## 十二、演示用关键词

水库调度、汛限水位、预泄、错峰调度、泄洪、下游预警、闸门操作、入库流量、出库流量、防洪库容、水库巡查。
