# III级响应现场处置要点（演示版）

## 一、适用场景

III 级防汛响应适用于较大范围强降雨、主要河道超警、局部工程险情、城区多点内涝、需要提前转移群众或跨部门协同处置的场景。III 级响应不是最高等级，但已进入实战状态，现场处置应做到“先控风险、再查原因、边处置边报告”。

## 二、总体目标

1. 防止一般险情扩大为重大险情。
2. 防止局部积水演变为人员伤亡事件。
3. 防止工程调度、交通管制、群众转移之间脱节。
4. 保持现场、值班室、会商组和指挥部信息同步。
5. 为 II 级或 I 级响应升级做好队伍、物资、安置点和指挥体系准备。

## 三、现场指挥组织

III 级响应启动后，重点区域应建立现场处置组。现场处置组建议包括：

| 小组 | 牵头单位 | 职责 |
|---|---|---|
| 综合协调组 | 属地政府或防指办 | 接收指令、协调队伍物资、汇总信息 |
| 工程技术组 | 水务或工程管理单位 | 判断堤防、水库、闸站、泵站风险 |
| 抢险处置组 | 应急、水务、消防、专业队伍 | 封堵、加固、抽排、排障、救援 |
| 转移安置组 | 属地政府、民政、公安 | 组织群众转移、安置、秩序维护 |
| 交通保障组 | 交通、公安、城管 | 封路绕行、清障、保通 |
| 通信电力组 | 通信、电力部门 | 保障现场通信、电源、照明 |
| 信息报送组 | 防指办、宣传部门 | 形成首报、续报、口径和影像资料 |

## 四、到场后 15 分钟动作

1. 确认现场安全边界，设置警戒线，禁止无关人员进入危险区域。
2. 确认人员安全，优先搜救被困人员和转移受威胁群众。
3. 核实险情位置、类型、规模、发展趋势和影响对象。
4. 向值班室回传定位、照片、短视频和初步判断。
5. 明确现场联系人、通信方式和信息报送频次。
6. 建立现场台账，记录每一道指令和反馈。

## 五、到场后 1 小时动作

| 任务 | 要求 |
|---|---|
| 风险研判 | 判断是否存在漫堤、管涌、倒灌、滑坡、触电、坍塌等风险 |
| 抢险布置 | 明确抢险点位、队伍人数、机械设备、物资需求 |
| 群众转移 | 对低洼区、危房、临河临坡、地下空间人员进行转移 |
| 交通管控 | 对积水路段、桥涵、下穿通道、临水道路实施封控 |
| 工程调度 | 根据会商意见调整闸门、泵站、水库泄量 |
| 信息续报 | 报送最新数据、处置进展、困难需求和下一步措施 |

## 六、重点场景处置

### 1. 河道超警

河道达到或超过警戒水位时，应加密巡查堤防、穿堤建筑物、涵闸、排口、低洼回水区。巡查频次建议不低于每 2 小时一次，重点险段可连续盯守。发现渗水、裂缝、滑坡、跌窝、管涌等异常，应立即标记位置、扩大警戒并报告工程技术组。

### 2. 堤防渗漏和管涌

处置原则是“临水截渗、背水导滤、反压稳坡、严禁盲堵”。对清水渗漏可先观察和导排；对浑水、带砂、出水量增大的管涌，应立即设置围井、反滤围堰或砂石反压。严禁在背水侧直接用不透水材料堵塞出水口。

### 3. 城区内涝

优先处置医院、学校、养老院、地下商场、地铁口、下穿通道、变电站、泵站、供水厂等重点对象。积水超过 30 厘米且仍上涨时，应限制车辆通行；积水进入地下空间或存在触电风险时，应立即封控并组织人员撤离。

### 4. 水库泄洪影响

水库泄洪前应完成下游预警叫应，通知沿河乡镇、施工营地、涉水景区、低洼住户、河道作业人员撤离。泄洪过程中应监测下游水位涨幅，严禁人员进入河道捕鱼、围观或通行。

### 5. 闸站泵站故障

出现机组跳闸、闸门卡阻、供电异常、泵站倒灌风险时，应立即切换备用机组或备用电源，通知电力和检修人员到场。无法恢复时，应评估临时抽排、移动泵车、沙袋封堵和交通封控方案。

## 七、群众转移要点

转移对象包括低洼区、临河临堤、山洪沟口、危旧房、地下空间、施工营地、孤寡老人、留守儿童、行动不便人员。转移应落实“四个明确”：明确转移责任人、转移路线、安置地点、返回条件。未经现场指挥确认，不得组织群众返回危险区域。

## 八、现场信息报送

现场首报应在到场后尽快完成，内容包括事件位置、险情类型、影响范围、人员安全、已采取措施、需要支援事项。续报应至少包含新数据、新变化、新措施和新需求。现场照片应标明时间、地点、拍摄方向和风险点名称。

## 九、升级建议

III 级响应期间出现以下情况，应建议升级：

1. 险情持续扩大，现有措施无法控制。
2. 水位继续上涨并接近保证水位。
3. 需要跨区县调派大规模队伍和物资。
4. 出现人员被困、伤亡或大范围转移。
5. 重要基础设施中断并影响抢险救援。

## 十、演示用关键词

III 级响应、现场处置、河道超警、堤防管涌、城区内涝、群众转移、交通封控、现场首报、抢险队伍、物资调拨、升级响应。
