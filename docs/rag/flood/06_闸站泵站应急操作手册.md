# 闸站泵站应急操作手册（演示版）

## 一、适用范围

本手册用于防汛演示知识库，适用于城市排涝泵站、河道节制闸、排水闸、涵闸、穿堤建筑物和临时移动泵车的应急运行。重点解决强降雨、外河高水位、城区积水、闸门故障、泵组故障、供电中断和倒灌风险等场景。

## 二、运行目标

1. 保证闸站泵站人员安全、设备安全和工程安全。
2. 在满足调度指令的前提下，尽快降低内河水位和城区积水。
3. 防止外河洪水倒灌、闸门误操作和泵站空转烧机。
4. 保障关键区域排涝：医院、学校、交通枢纽、地下空间、变电站、供水厂。
5. 做到操作有令、执行有记录、异常有报告、处置有闭环。

## 三、岗位分工

| 岗位 | 职责 |
|---|---|
| 站长或现场负责人 | 接收调度指令、决定现场处置、向值班室报告 |
| 闸门操作员 | 执行闸门启闭、复核闸位、观察水流状态 |
| 泵组操作员 | 启停泵组、监测电流、电压、振动、温度 |
| 电气保障员 | 检查配电柜、备用电源、发电机、线路安全 |
| 巡查员 | 巡查进出水口、拦污栅、堤身、涵洞和周边积水 |
| 记录员 | 填写运行日志、调度令、故障单和影像资料 |

## 四、启动条件

出现以下情况，应进入应急运行状态：

1. 站前或内河水位达到预警水位并持续上涨。
2. 城区低洼点积水超过 20 厘米，且降雨仍在持续。
3. 外河水位高于内河水位，存在倒灌风险。
4. 上级下达排涝、关闸、开闸、预排或限排指令。
5. 泵组、闸门、供电、通信、拦污栅出现异常。
6. 医院、学校、地下空间、交通枢纽等重点区域提出排涝需求。

## 五、开机前检查

泵组启动前必须检查：

1. 进水池水位满足开机条件，避免空转。
2. 拦污栅无大面积堵塞，进水流态正常。
3. 出水通道畅通，外河水位允许排水。
4. 电压、电流、相序、接地和保护装置正常。
5. 机组轴承、冷却、润滑、密封状态正常。
6. 备用机组、备用电源、移动照明和排水工具可用。

## 六、泵站排涝操作

| 步骤 | 操作要求 |
|---|---|
| 接令 | 核对调度指令、目标水位、启动机组数量 |
| 试运 | 单机短时试运行，确认无异常振动、异响、过热 |
| 启动 | 按先小后大、逐台启动原则投入机组 |
| 监测 | 每 15 分钟记录水位、电流、电压、温度、振动 |
| 调整 | 根据水位变化和外河承泄能力调整机组数量 |
| 停机 | 水位降至控制值后逐台停机，避免水锤 |
| 复核 | 检查设备状态，清理拦污栅，填写运行记录 |

## 七、闸门操作

闸门启闭应执行双人复核制度。操作前核对闸门编号、调度令、开度目标、启闭方向、上下游水位和现场安全条件。操作过程中应缓慢启闭，观察水流、闸门振动、启闭机电流和闸槽异物。发现卡阻、异响、闸门倾斜、启闭机过载时，应立即停止操作并报告。

外河水位高于内河水位时，原则上关闭排水闸，防止倒灌；确需排水时，应经会商确认并采取泵排或临时防倒灌措施。

## 八、常见故障处置

| 故障 | 表现 | 处置要点 |
|---|---|---|
| 泵组跳闸 | 保护动作、机组停运 | 查明电气或机械原因，切换备用机组 |
| 拦污栅堵塞 | 进水水位异常、流量下降 | 停机或减载后清污，防止人员落水 |
| 闸门卡阻 | 闸位不变、电流升高 | 停止操作，检查异物和启闭机 |
| 外河倒灌 | 出水口回流、内河水位反涨 | 关闸、停泵、设临时挡水 |
| 供电中断 | 全站失电 | 启动备用电源，联系电力抢修 |
| 通信中断 | 无法接收调度令 | 启用备用手机、卫星电话或现场联络员 |

## 九、安全要求

1. 暴雨、雷电、夜间作业必须佩戴救生衣、安全帽和照明设备。
2. 严禁人员单独进入泵房、闸室、涵洞和临水区域。
3. 清理拦污栅应先停机或降低流速，设置防坠落措施。
4. 电气设备进水时不得盲目送电，应由电气专业人员确认。
5. 现场积水可能带电时，应立即封控并通知电力部门。

## 十、信息报送

泵站运行报送内容包括：站名、内外河水位、开机台数、总流量、累计排水量、设备状态、故障情况、需支援事项。闸站报送内容包括：闸门编号、开度、上下游水位、过闸流量、启闭时间、异常情况。

## 十一、停机与恢复

强降雨结束后，应根据调度指令逐步停机。停机后检查机组温度、轴承、密封、电缆、配电柜、拦污栅和泵房积水。对故障设备应挂牌停用，形成维修单，不得带病投入下一轮强降雨。

## 十二、演示用关键词

闸站、泵站、排涝、倒灌、闸门启闭、泵组跳闸、备用电源、移动泵车、拦污栅、内河水位、外河水位、调度指令。
