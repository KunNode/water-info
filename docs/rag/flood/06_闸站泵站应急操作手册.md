# 闸站/泵站应急操作手册（通用版草案）

## 适用范围

本手册适用于承担防洪、排涝、引排调度任务的闸站、泵站、雨水泵站及相关附属设施。手册按“开机前检查—运行中监控—异常工况处置—停机恢复—信息报送”五段式编排，便于按段切块入库。城市排水泵站运行维护地方标准通常将水泵机组、电气设备、自控系统、进出水设施、安全管理、档案与信息化管理纳入统一要求；泵站一旦发生突发险情，其首报字段应纳入防汛险情报送体系。

## 岗位与职责

| 岗位 | 主要职责 |
|---|---|
| 值班负责人 | 接令、组织启停、统筹安全和报送 |
| 电气值守员 | 电源、配电、启停柜、备用电源检查 |
| 机泵值守员 | 水泵、阀门、闸门、油温、振动、噪声监测 |
| 巡检员 | 进出水口、拦污栅、闸室、厂房、围堰巡查 |
| 信息员 | 留存启停时间、参数、异常和报送记录 |

## 开机前检查

核对调度指令、适用工况、启泵条件和安全边界值。

检查主电源、备用电源、UPS、发电机、通信链路是否正常。

检查泵机、启闭机、闸门、阀门、液压系统、润滑和冷却状态。

清理拦污栅和进水口漂浮物，确认进出水通道畅通。

检查厂房、配电室、地下空间、防倒灌和防触电措施是否到位。

复核现场人员分工、警戒区设置、应急停机和人工切换方案。

## 运行中监控

建议至少监控以下参数：{{前池/集水池水位}}、{{出水侧水位}}、{{单机电流电压}}、{{机组振动/温度}}、{{闸门开度}}、{{运行台数}}、{{累计运行时长}}、{{厂房进水和渗漏情况}}。当出现倒灌、负荷异常、泵机空转、振动升高、异响、异味、温度异常或电气告警时，应立即执行异常工况处置卡。

## 异常工况处置

| 异常情形 | 首要动作 | 后续处置 |
|---|---|---|
| 主电源中断 | 切换备用电源或启动发电机 | 报告调度并核验关键负荷 |
| 泵机异常振动/高温 | 立即减载或停机检查 | 排查轴承、叶轮、润滑和异物 |
| 闸门无法启闭 | 采取手动/备用启闭方案 | 设置警戒并上报设备故障 |
| 地下空间倒灌 | 立即封控、断电、转移人员 | 启动临时抽排和挡水措施 |
| 人员触电或伤害 | 先救人、后处置设备 | 启动医疗救援并保护现场 |

## 报送要求

涉及泵站、涵闸突发险情的首报内容，建议至少包括：工程名称、位置、管理单位、出险时间、出险部位、险情类型、上下游水位、设备状态、影响范围、已采取措施、现场联系人。紧急情况下先电话报送，随后补充书面材料和图片。

## 风险提示

闸站/泵站处置最忌“只盯启停、不盯电气安全”“只看水位、不看人员暴露风险”。凡涉及地下空间、带电设备、人工下井、夜间抢修、暴雨中清障作业，必须实行双人以上作业、现场警戒、断电挂牌和应急撤离预案。设备厂家手册、本地运行规程和调度授权表优先于本通用草案。
