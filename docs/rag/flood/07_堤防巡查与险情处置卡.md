# 堤防巡查与险情处置卡（演示版）

## 一、适用范围

本处置卡用于防汛演示知识库，适用于河道堤防、水库坝体、穿堤建筑物、涵闸、泵站出水口、险工险段和历史隐患点的巡查检查与初期险情处置。处置卡强调“巡查发现早、报告定位准、抢护方法对、人员安全优先”。

## 二、巡查启动条件

出现以下情况，应启动或加密堤防巡查：

1. 河道水位达到警戒水位或上涨较快。
2. 连续强降雨导致堤身、坝坡长期浸泡。
3. 上游水库泄洪或下游顶托导致水位突涨。
4. 历史险工险段、穿堤建筑物、软基堤段处于高水位。
5. 群众报告渗水、裂缝、塌陷、异响、浑水冒出。
6. 防汛响应启动后，指挥部要求加密巡查。

## 三、巡查频次

| 水位或状态 | 巡查频次 |
|---|---|
| 常态汛期 | 每日不少于 1 次 |
| 接近警戒水位 | 每 4 小时不少于 1 次 |
| 超过警戒水位 | 每 2 小时不少于 1 次 |
| 重要险工险段 | 连续盯守或每 1 小时巡查 |
| 发生险情 | 专人值守，实时报告变化 |

夜间巡查应两人以上同行，携带照明、救生衣、对讲机、定位设备和警示标志。

## 四、巡查重点

| 部位 | 检查内容 |
|---|---|
| 迎水坡 | 浪坎、滑坡、裂缝、塌陷、护坡损坏 |
| 堤顶 | 纵向裂缝、横向裂缝、沉陷、车辆碾压 |
| 背水坡 | 渗水、散浸、脱坡、鼓包、湿软 |
| 堤脚 | 管涌、冒水、冒砂、浑水、跌窝 |
| 穿堤建筑物 | 涵闸渗漏、接触冲刷、闸门闭合不严 |
| 险工险段 | 历史险点复发、护岸冲刷、近岸坍塌 |

## 五、险情识别

| 险情类型 | 典型表现 | 风险判断 |
|---|---|---|
| 渗水 | 背水坡潮湿或清水渗出 | 需观察水量和浑浊度 |
| 管涌 | 堤脚冒水冒砂，水流带泥沙 | 高风险，可能发展为溃堤 |
| 漏洞 | 集中出流，水量明显 | 高风险，需快速封堵和反滤 |
| 滑坡 | 坡面裂缝、下滑、隆起 | 可能削弱堤身稳定 |
| 裂缝 | 堤顶或坡面开裂 | 横向裂缝风险更高 |
| 漫顶 | 洪水接近或越过堤顶 | 极高风险，应立即加高抢护和转移 |
| 跌窝 | 局部塌陷、坑洞 | 可能与内部淘刷有关 |

## 六、处置原则

1. 先报告、先警戒、先避险，再组织抢护。
2. 临水侧以截流、防浪、压盖为主，背水侧以导滤、反压、排水为主。
3. 管涌、漏洞等出水点严禁直接用黏土或不透水材料硬堵。
4. 抢护方案应由工程技术人员确认，现场可先采取临时控制措施。
5. 险情未稳定前，受威胁区域群众不得返回。

## 七、典型险情处置卡

### 1. 管涌

处置步骤：

1. 设置警戒，禁止人员靠近出水口。
2. 标记出水点位置，记录水色、水量、冒砂情况。
3. 立即报告值班室和工程技术组。
4. 在背水侧设置反滤围井，材料可用砂石、碎石、土工布、编织袋。
5. 保持出水通畅，降低水流带砂能力。
6. 加密观察围井水位、浑浊度和周边新冒点。

### 2. 渗水散浸

处置步骤：

1. 清理坡面杂物，确认渗水范围。
2. 开挖浅沟导排，避免水流冲刷坡脚。
3. 铺设反滤材料，必要时砂袋反压。
4. 持续观察是否变浑、变大或形成集中出流。

### 3. 滑坡

处置步骤：

1. 在滑坡体上下游设置警戒。
2. 减少堤顶荷载，禁止车辆通行。
3. 背水坡滑动时可采用削坡减载、坡脚压重、覆盖防雨。
4. 临水坡滑动时应防浪护坡，必要时抛石或袋装土压脚。

### 4. 漫顶

处置步骤：

1. 立即报告并建议转移受威胁区域人员。
2. 使用编织袋、土袋、移动挡水板加高子堤。
3. 临水侧铺设防渗布，背水侧设置排水沟。
4. 若水位继续上涨且抢护无效，应组织人员撤离。

## 八、巡查记录字段

| 字段 | 要求 |
|---|---|
| 巡查时间 | 精确到分钟 |
| 巡查人员 | 至少两人签名 |
| 巡查河段 | 堤名、桩号、岸别 |
| 水位情况 | 当前水位、涨落趋势 |
| 发现问题 | 类型、位置、规模、照片 |
| 处置措施 | 警戒、报告、抢护、转移 |
| 后续建议 | 加密巡查、专家会商、物资支援 |

## 九、演示用关键词

堤防巡查、险工险段、管涌、渗水、漏洞、滑坡、裂缝、漫顶、反滤围井、砂袋反压、穿堤建筑物、堤脚冒砂。
